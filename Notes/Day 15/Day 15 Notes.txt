ğŸ“‚ Common Modes:
Mode	Meaning
'r'	Read (default), file must exist
'w'	Write, creates new file or overwrites
'a'	Append, adds to end, creates if not exist
'r+'	Read and Write
'w+'	Write and Read (clears existing)

ğŸ“– 2. Reading Files

# Read entire file content
file = open("sample.txt", "r")
content = file.read()
print(content)
file.close()

# Read one line at a time
file = open("sample.txt", "r")
line1 = file.readline()
line2 = file.readline()
file.close()

# Read all lines into a list
file = open("sample.txt", "r")
lines = file.readlines()   # ['line1\n', 'line2\n', ...]
file.close()

âœ… 4. Using with Statement (Best Practice)

with open("sample.txt", "r") as file:
    data = file.read()
    print(data)
# File is auto-closed here

with open("new.txt", "w") as f:
    f.write("First line\nSecond line")
