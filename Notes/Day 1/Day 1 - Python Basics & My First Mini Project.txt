🗓️ Day 1 - Python Basics

🔹 print(): Used to display output
🔹 input(): Used to take user input (always returns a string)
🔹 Variable: A container for storing data (e.g., name = "Arka")
🔹 Data Types: str (text), int (whole numbers), float (decimal), bool (True/False)
🔹 Type Conversion: Use int(), float(), str() to change types


🧮 Python Operators
+  Addition         → 5 + 3 = 8  
-  Subtraction      → 10 - 6 = 4  
*  Multiplication   → 2 * 4 = 8  
/  Division         → 9 / 3 = 3.0  
// Floor Division   → 9 // 2 = 4  
%  Modulus (Remainder) → 9 % 2 = 1  
** Exponentiation   → 2 ** 3 = 8

📦 Type Checking:
type(5) → int
type(5.0) → float
type("hello") → str

🧪 Type Conversion:
int("5") → 5  
float("5.6") → 5.6  
str(5) → "5"


🔠 String Operations in Python

+  Concatenation: "Hello " + "World" → "Hello World"
*  Repetition: "Ha" * 3 → "HaHaHa"
[] Indexing: "Arka"[0] → "A"
[:] Slicing: "Arka"[1:3] → "rk"
len(): Length of a string → len("Hello") = 5
.lower() / .upper(): Case conversion
.strip(): Removes whitespace
.replace(): Replace parts of a string
[::-1] To reverse the sentence or word (ex: sentence[::-1])


🧠 Python Conditional Statements

if condition:
    # do something
elif another_condition:
    # do something else
else:
    # fallback
