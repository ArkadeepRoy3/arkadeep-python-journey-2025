🔹 1. String Immutability & Indexing
Strings cannot be changed in-place (immutable)

You can index them like lists: s[0], s[-1]

Reverse: s[::-1]

🔹 2. Case Transformations
Method	Description
s.lower()	All lowercase
s.upper()	All uppercase
s.title()	Capitalizes first letter of each word

🔹 3. Cleanup Methods
Method	Description
s.strip()	Removes leading/trailing spaces
s.replace(a, b)	Replaces all a with b
s.split()	Splits string by whitespace or delimiter
' '.join(list)	Joins list into a single string

🔹 4. Validation Methods
Method	Description
s.isalpha()	All letters
s.isalnum()	Letters + numbers (no symbols)
s.startswith(x)	Checks if string starts with x

✅ Common in login systems, input validation, and user authentication
